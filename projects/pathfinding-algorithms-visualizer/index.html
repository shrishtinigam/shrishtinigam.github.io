Pathfinding Algorithms Visualizer (SFML)

A C++ application that visually demonstrates four fundamental pathfinding algorithms:

  • Breadth-First Search (BFS)
  • Depth-First Search (DFS)
  • Dijkstra's Algorithm
  • A* (A-Star) Algorithm

Built using the SFML graphics library, this tool allows users to interactively explore how these algorithms traverse a grid to find the shortest path between two points.

Features

  • Interactive Grid: Click to set start and end points.
  • Algorithm Selection: Choose between BFS, DFS, Dijkstra, and A*.
  • Wall Placement: Block paths by clicking to add walls.
  • Real-Time Visualization: Watch the algorithms in action step-by-step.

Added DFS visualization

  • Added another map and window that shows visualization for the DFS pathfinding algorithm.
  • DFS uses stack data structure to perform its search.
  • The time complexity of the DFS algorithm is O(V+E), where V is the number of vertices and E is the number of edges in the graph

Optimized A Star code

  • Instead of using a vector to store the visited cells values, I used a priority queue.
  • In particular, I implemented a min heap data structure.
  • This reduced the search time complexity from O(n) to O(1)
  • Previously, A star was performing just as good or worse than BFS/Dijsktra's.
  • Now, it performs much faster. However, this ofcourse depends on the case and the given map.
  • In order to create a min heap, I used the following template -

cpp std::priority_queue <std::pair<int, gbl::Position<>>> astar_path_minheap;

  • Here, in the first 'int', I store the f score of the cell, multiplied by (-1). This way, the cells with the minimum f scores rise up to the top.

Installation

Prerequisites

Ensure you have the following installed:

  • SFML 2.5+: SFML Downloads
  • C++ Compiler: Compatible with C++11 or later

Building the Project

  1. Clone the repository:
git clone https://github.com/shrishtinigam/pathfinding_algorithms_visualizer_sfml.git
cd pathfinding_algorithms_visualizer_sfml
  1. Compile the project:
g++ -std=c++11 -o pathfinder Main.cpp BFS.cpp DFS.cpp Dijkstra.cpp AStar.cpp -lsfml-graphics -lsfml-window -lsfml-system
  1. Run the application:
./pathfinder

Usage

  • Left Click: Place walls on the grid
  • Right Click: Set start and end points
  • Press 'B': Run BFS
  • Press 'D': Run DFS
  • Press 'I': Run Dijkstra
  • Press 'A': Run A*
